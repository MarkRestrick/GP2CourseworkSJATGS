# Camera Controls:
W: Move forwards

S: Move backwards

A: Move left

D: Move right

P: Toggle debug mode

Space Bar: Move up if in debug mode otherwise jump

Left Shift: If in debug mode move down

1 - 5: Toggle post processing effects
